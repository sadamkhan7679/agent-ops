# Security Policy

## Reporting a Vulnerability

Please do not open a public GitHub issue for security reports.

Report vulnerabilities privately to:

- `security@agentops.dev`

Include:

- A description of the issue
- Steps to reproduce
- Impact assessment
- Any suggested remediation

We will acknowledge receipt, investigate, and coordinate remediation before any
public disclosure.

## Scope

This policy covers:

- The Next.js showcase application
- Local content parsing and rendering logic
- Repository automation and generated documentation

Third-party services and dependencies should be reported according to their own
security policies where appropriate.
